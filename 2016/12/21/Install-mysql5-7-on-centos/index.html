发表于 2016-12-21 | 分类于 运维 |

The step of Installing Mysql5.7.17

楼主只介绍一种rpm包安装的方式（相对比较简单):

  1. 下载合适的rpm， download mysql’s rpm

  2. 根据你的Linux系统执行如下命令（楼主的Linux 系统是centos7 ）：yum localinstall mysql57-community-release-el7-{version-number}.noarch.rpm

  3. 选择一个已经发布的GA series安装， 默认是mysql 最新的版本5.7， 可以使用命令验证你的安装包， 命令如下yum repolist enabled | grep mysql

  4. 安装mysql server: yum install mysql-community-server

  5. 启动mysql server: service mysqld start

  6. 你可以使用命令验证mysql server 是否安装并启动成功，命令如下：service mysqld status， 如果结果出现 类似 Active: active (running) 或者 mysqld (pid 3066) is running.这样的字样 即说明安装成功！

  7. 至此，

    • mysql server 已经完成初始化；

    • data 目录下 已经产生一个SSL 证书和 key 文件；

    • validate_password插件已经被安装并且可以使用；

    • 一个超级用户’root’@localhost 也已经被创建。这个超级用户的密码被设置并且存在于error log file中， 可以用如下命令查看超级用户的初始密码：grep 'temporary password' /var/log/mysqld.log

    • 得到这个初始化密码后，你就可以以root 身份登陆了， 如果你想重置密码可以使用如下命令：mysql -uroot -p（以root 用户登陆） ， 在等待输入的地方 输入你的初始化密码， 然后利用mysql 命令：ALTER USER 'root'@'localhost' IDENTIFIED BY 'MyNewPass4!';重置即可。

      ​

  8. mysql server 安装完成。

上面的步骤介绍全部来自于mysql 官方文档
