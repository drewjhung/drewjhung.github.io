mysql使用中问题错误汇总

  • Error code 1130

    原因：客户端ip 不被允许利用此账号连接server， 需要把该客户端ip 加入该账户对应的可远程登陆的信任列表

    解决办法：执行如下2条命令：

    GRANT ALL ON *.* to root@'%' IDENTIFIED BY 'pass';

    FLUSH PRIVILEGES;
